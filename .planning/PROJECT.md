# TESDA Skills-to-Jobs Matcher

## Project Code
TESDA-MATCHER

## Overview
AI-powered vocational course and job recommendation platform for Filipino youth (15-24). Helps out-of-school youth discover TESDA courses and career opportunities based on their skills, interests, and location.

## Tech Stack
- React 19 + TypeScript
- Tailwind CSS 4
- Express.js backend
- Vite bundler
- Fireworks AI API (Kimi models)
- Bilingual: Taglish and English

## Target Audience
Out-of-school youth (15-24) in the Philippines seeking vocational training and employment opportunities.

## Key Features (Planned)
1. AI-powered course matching
2. Job market recommendations with salary data
3. Interactive assessment flow
4. Real-time chat counselor
5. Mobile-first responsive design

## Current Status
Initial development phase. Core infrastructure and course matching exist. Job matching feature planned.
