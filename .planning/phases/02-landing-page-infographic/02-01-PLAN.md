## Design Decisions (Locked)

From sketch wrap-up and project decisions:
- **Variant B (Step-by-Step Journey)** is the winner — story-driven engagement for users who feel lost about their career path
- Landing CTAs: Primary "Begin Your Journey" + secondary Chat/Browse — action-oriented language, multiple entry points
- Typography: Plus Jakarta Sans (display) + Inter (body) — modern, bold headlines + clean readable body
- Spacing: 4px base, generous mobile padding — touch-friendly, breathable mobile layouts
- Radius: 12-16px cards, 24px CTAs, 9999px pills — friendly, approachable shapes
- Transitions: 0.2s UI, 0.4s state changes — smooth but not slow, keeps mobile feel snappy
- Colors: Blue-600/blue-700 primary, slate for text, emerald for success accents (matching existing app)
- Lucide icons throughout — no emoji in production code
- Bilingual: Taglish and English support via `lang` prop

## Existing Component Patterns (from App.tsx and Navbar.tsx)

- Tailwind classes: `rounded-3xl`, `bg-gradient-to-r`, `shadow-2xl`, `font-display`, `space-y-8`, `max-w-7xl`
- Color scheme: blue-600/700 primary, slate-800/900 text, white surfaces, indigo accents
- Animation: `animate-fade-in` custom class, staggered delays (`stagger-1`, `stagger-2`, etc.)
- Cards: `bg-white rounded-3xl border border-slate-200 p-8 shadow-lg card-hover`
- Buttons: `rounded-2xl bg-white px-6 py-4 text-base font-bold text-blue-700 shadow-xl hover:bg-blue-50 hover:shadow-2xl hover:-translate-y-1 transition-all`
- Secondary buttons: `rounded-2xl bg-white/15 border border-white/30 px-6 py-4 text-base font-bold text-white hover:bg-white/25 hover:-translate-y-1 transition-all`
- Icons from lucide-react: Sparkles, MessageSquare, BookOpen, etc.
- Bilingual: `lang === "fil"` ? Taglish : English
- Mobile-first: `sm:`, `md:`, `lg:` breakpoints used throughout
