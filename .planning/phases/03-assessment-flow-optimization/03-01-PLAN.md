## Objective
Create a dedicated `AssessmentWizard` component that transforms the monolithic single-page form into a clear sequential multi-step wizard with step navigation, progress indicators, and smooth transitions. All existing form fields and logic are preserved; only the UX flow changes.

## Context
- `src/App.tsx` lines 465-713 contain the current monolithic form with all fields visible at once (age slider, education select, region/province selects, interest tags, skill tags, career goal textarea, submit button)
- `src/types.ts` contains `UserProfile`, `MatchingResult`, `RecommendationCourse`
- Existing handlers in App.tsx: `handleSubmitProfile`, `toggleInterestTag`, `toggleSkillTag`, `handleAddCustomInterest`, `handleAddCustomSkill`, `handleRegionChange`
- Existing state: `age`, `education`, `selectedRegion`, `selectedProvince`, `customInterests`, `customSkills`, `interestInput`, `skillInput`, `careerGoal`, `isMatching`, `matchResult`, `matchError`, `lang`
- Bilingual support via `lang` prop ('fil' | 'en')
- Design system: Tailwind CSS, rounded-3xl cards, gradient buttons, Lucide icons, card-hover classes, animate-fade-in
