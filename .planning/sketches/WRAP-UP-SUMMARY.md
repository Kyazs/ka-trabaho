## Design Direction

Mobile-first redesign for a TESDA skills-to-jobs matcher app targeting Filipino out-of-school youth (ages 15-24). The design replaces the immediate form-dump landing page with a story-driven infographic that builds trust, and creates a clear sequential assessment flow (form → loading → results) rather than showing everything at once.

**Key principles:**
- Encouraging, non-judgmental tone (Taglish/English bilingual)
- Story-driven onboarding: "Your Path to Success" journey
- Single-focus states: never show form + results simultaneously
- Progress transparency: animated loading steps explain what the AI is doing
- Trust through data: stats (300+ courses, ₱160/day, NC II, 95% placement) before asking for input

## Key Decisions

| Area | Decision | Why |
|------|----------|-----|
| **Landing** | Step-by-step journey infographic | Motivates users who feel lost about their career path |
| **Landing CTAs** | Primary "Begin Your Journey" + secondary Chat/Browse | Action-oriented language, multiple entry points |
| **Assessment flow** | Full page replacement | Cleanest mobile experience, single focus at a time |
| **Loading state** | Spinner + 4 progress steps | Makes wait feel productive and transparent |
| **Results** | AI summary + 2 course cards with match % | Top matches with clear reasoning, not overwhelming |
| **Typography** | Plus Jakarta Sans (display) + Inter (body) | Modern, bold headlines + clean readable body |
| **Spacing** | 4px base, generous mobile padding | Touch-friendly, breathable mobile layouts |
| **Radius** | 12-16px cards, 24px CTAs, 9999px pills | Friendly, approachable shapes |
| **Transitions** | 0.2s UI, 0.4s state changes | Smooth but not slow, keeps mobile feel snappy |
