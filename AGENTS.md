## Project Overview
- React + Tailwind CSS application
- TESDA skills-to-jobs matcher for Filipino youth (15-24)
- AI-powered course and job recommendation
- Bilingual: Taglish and English
- Mobile-first design direction validated through sketches

## Design Context
- **Register:** product (design serves usability and task completion)
- **North Star:** "The Open Doorstep" — no barriers, the path is always lit
- **Palette anchors:** Depth Blue (#0F3D91) + Sunshine Gold (#FCD116) from the Philippine flag
- **Key rules:** Two-Anchor Rule (only 2 saturated non-semantic colors per screen), Neutral-90 Rule (90% surface is neutral), Rest-Flat Rule (no shadows at rest), Weight-Gap Rule (200+ weight gap between heading and body)
- **Anti-references:** Government portal stiffness AND corporate SaaS dashboard
- **Full spec:** See `PRODUCT.md` (strategy) and `DESIGN.md` (visual system)
